Titulo del proyecto

MicroMaster


Descripcion

MicroMaster es un proyecto de desarrollo web que consiste en una calculadora de insumos para diferentes sitios de trabajos como cocinas, barismo, enfermeria, etc. La aplicacion basicamente es poder calcular onzas, litros, mililitros, macros, etc. La aplicacion permite al usuario establecer claculos precicios para disminuir perdidas tanto economicas como de insumos.
